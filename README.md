# 🛒 FreshBasket — Online Grocery Shopping Cart

A responsive, interactive shopping cart built for **FreshBasket**, an online grocery store.  
Built as part of the **Cedcoss Technologies Round 2 Assignment**.

---

## 🌟 Features

- Browse **12 grocery products** with emoji, unit info, and price
- **Add items to cart** with a single click
- **Change quantity** (+ / −) directly in the cart panel
- **Remove items** by reducing quantity to zero (trash icon)
- **Live total** updates automatically (subtotal + delivery fee)
- **Free delivery** on orders above ₹500 🎉
- **"Your cart is empty"** message with friendly prompt when cart has no items ✅ *(bonus)*
- **Clear all** button to reset the cart
- **Checkout simulation** — places the order and resets the cart
- Sticky cart panel on desktop so you always see your cart while browsing
- Toast notifications when items are added or removed
- Fully **responsive** — works on mobile and desktop

---

## 🚀 How to Run

No installation needed. This is a **pure HTML/CSS/JavaScript** single-file project.

1. Clone or download this repository
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge)
3. Start shopping! 🥦🍎🥛

```bash
git clone https://github.com/YOUR_USERNAME/ecommerce-task-cedcoss.git
cd ecommerce-task-cedcoss
# Then open index.html in your browser
```
